## 1. アプリ概要

**Life Explorer** は「誰かのライフスタイルを借りて、まだ知らない自分に出会おう」をコンセプトにした、ルーティン探索・実践アプリです。

### コア機能

| 機能 | 説明 |
|------|------|
| **マイルーティン** | 自分の一日のタイムラインを作成・管理する |
| **一日を借りる** | 他のユーザーやペルソナのルーティンを試す |
| **探索** | 78件以上のペルソナカタログ・コミュニティ投稿を発見する |
| **朝のリマインダー** | Web Pushで毎朝指定時刻に通知を受け取る |

### 技術スタック

| レイヤー | 技術 |
|---------|------|
| フロントエンド | Next.js 15, React, Tailwind CSS |
| バックエンド | Supabase (PostgreSQL + Auth + RLS) |
| ホスティング | Vercel (Hobby Plan) |
| プッシュ通知 | Web Push API + VAPID + Vercel Cron |
| AI生成 | Claude API / OpenAI API (任意) |

## 10. データモデル

### Supabase テーブル構成

```
auth.users （Supabase管理）
  └── user_settings       # ユーザー設定（1行/ユーザー）
  └── user_tasks          # タスク記録（∞行）
  └── public_routines     # 公開ルーティン
  └── likes               # いいね
  └── push_subscriptions  # 通知サブスクリプション
```

### user_tasks

| カラム | 型 | 説明 |
|--------|-----|------|
| id | UUID | PK |
| user_id | UUID | FK → auth.users |
| task_date | DATE | 日付 |
| time | TEXT | 時刻（HH:MM） |
| end_time | TEXT | 終了時刻（任意） |
| title | TEXT | タスク名 |
| thought | TEXT | メモ |
| type | TEXT | nature / mind / work |
| created_at | TIMESTAMPTZ | 作成日時 |

### public_routines

| カラム | 型 | 説明 |
|--------|-----|------|
| id | UUID | PK |
| user_id | UUID | FK → auth.users |
| display_name | TEXT | 表示名 |
| title | TEXT | ルーティンタイトル |
| category | TEXT | カテゴリ |
| routine | JSONB | タスク配列 |
| likes_count | INTEGER | いいね数 |
| is_published | BOOLEAN | 公開フラグ |

## 11. インフラ構成

```
ユーザー
  │
  ▼
Vercel（Next.js）
  ├── app/ ── フロントエンド全体
  ├── api/push/ ── Web Push API
  └── api/cron/ ── 朝のリマインダー Cron（毎時実行）
  │
  ▼
Supabase（PostgreSQL + Auth）
  ├── RLS: ユーザーは自分のデータのみアクセス可
  └── Auth: メール+パスワード認証
```

### 環境変数（Vercel）

| 変数名 | 用途 |
|--------|------|
| `NEXT_PUBLIC_SUPABASE_URL` | Supabase プロジェクト URL |
| `NEXT_PUBLIC_SUPABASE_ANON_KEY` | Supabase 公開キー |
| `NEXT_PUBLIC_VAPID_PUBLIC_KEY` | Web Push 公開キー |
| `VAPID_PRIVATE_KEY` | Web Push 秘密キー |
| `VAPID_SUBJECT` | Web Push 連絡先 |
| `CRON_SECRET` | Cron エンドポイント保護キー |
